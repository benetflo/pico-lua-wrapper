UART Module UART serial communication functions from C library

hardware.uart.write_string (uartx, string)
Sends a string from selected UART instance Binding to: luauartwrite_string (C function)

Parameters:

  • uartx UART instance number (UART0, UART1)
  • string message to send

Usage:

    hardware.uart.write_string("UART0", "Hello World!")
hardware.uart.set_config (uartx, data_bits, stop_bits, parity, cts, rts)
Configures the UART peripheral

Parameters:

  • uartx UART instance number (UART0, UART1)
  • data_bits Number of data bits (5-8)
  • stop_bits Number of stop bits (1 or 2)
  • parity Parity mode (NONE, EVEN, ODD)
  • cts Enable CTS flow control
  • rts Enable RTS flow control

Usage:

    hardware.uart.set_config("UART0", 8, 1, "NONE", false, false)
