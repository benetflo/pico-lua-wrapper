Module hardware.gpio

GPIO Module Hardware GPIO control functions from C library

Functions

set_output (pin, state)
Sets GPIO pin to HIGH or LOW. Init is done automatically.

Parameters:

  • pin GPIO pin number (0-22, 26-28)
  • state "HIGH" or "LOW"

Usage:

    hardware.gpio.set_output(16, "HIGH")
set_input (pin)
Configure GPIO pin as input with pull down as default. Init is done automatically.

Parameters:

  • pin GPIO pin number (0-22, 26-28)
get_input (pin)
Reads value from given GPIO pin

Parameters:

  • pin GPIO pin number (0-22, 26-28)

Returns:

    0 if LOW, 1 if HIGH

Usage:

    if hardware.gpio.get_input(15) == 0 then
    print("Pin is LOW")
end
set_pulls (pin, pupd)
Set pull-up or pull-down resistor

Parameters:

  • pin GPIO pin number (0-22, 26-28)
  • pupd "UP", "DOWN" or "NONE"

Usage:

    hardware.gpio.set_pulls(16, "UP")
